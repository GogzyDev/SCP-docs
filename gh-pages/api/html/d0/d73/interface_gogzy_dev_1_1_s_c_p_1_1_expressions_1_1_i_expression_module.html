◆ BuildExpression()

Expression BuildExpression ( string token,
Expression[] subExpressions )

Build the Expression node. The parser will supply sub-expressions for operators or functions. For an unary operator, you'll get 1 sub-rxpression. For a binary operator, you'll get 2 sub-expressions. For a function call with N arguments, you'll get N sub-expressions, etc.
